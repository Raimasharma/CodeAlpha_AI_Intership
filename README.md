# CodeAlpha_AI_Intership

This repository contains the Artificial Intelligence internship tasks completed for CodeAlpha.

Task 1: Language Translation Tool

- Developed using Python and Streamlit
- Supports multiple language translations

Task 2: FAQ Chatbot

- Developed using NLP techniques
- Uses TF-IDF Vectorization and Cosine Similarity

Task 3: Object Detection

- Developed using YOLOv8 and OpenCV
- Detects objects in real-time using webcam

Technologies Used

- Python
- Streamlit
- NLTK
- Scikit-Learn
- OpenCV
- YOLOv8
